# Contacts Search

Search, create, edit, and delete your Apple Contacts directly from Raycast — with a rich split-pane detail view showing photos, phone numbers, emails, addresses, birthday, job title, and notes.

## Features

- **Instant search** — alphabetically-sectioned list; search by name or company
- **Rich detail panel** — always-visible right pane with photo, all phones, emails, addresses, birthday, job title, and notes
- **Create contacts** — `⌘N` opens a form to add a new contact to your default address book
- **Edit contacts** — `⌘E` opens a pre-populated form to update name, job, phone, email, or notes
- **Delete contacts** — `⌃X` with a destructive confirmation dialog
- **Quick actions** — compose email, call, open in Contacts, copy phone/email/name

## Setup

No API keys or configuration required. On first run, macOS will prompt you to grant Raycast access to Contacts. Click **OK** to allow it.

## Commands

| Command | Description |
|---|---|
| Search Contacts | Search and browse all Apple Contacts |

## Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `⌘N` | Create new contact |
| `⌘E` | Edit selected contact |
| `⌃X` | Delete selected contact |
| `⌘⇧P` | Copy primary phone number |
| `⌘⇧N` | Copy name |
| `⌘O` | Open contact in Contacts.app |
| `⌘R` | Refresh contacts list |
